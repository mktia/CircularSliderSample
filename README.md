# A sample app using circular slider controller
It looks like bedtime settings in iOS.

## Requirement

- React Native 0.57.1
- expo (expo-cli 2.11.9)

## Usage

```
$ npm install

# for Android
$ npm run android

# for iOS
$ npm run iOS
```
